Atom _[js2coffee]_
==================
Transpile JavaScript to [CoffeeScript] in [Atom] with [js2coffee].

[Another] plugin for _[atom-transpile]_.

Install
-------
`apm install transpile-js2coffee` or search “js2coffee” under Packages within Atom.
